# ac-my-storybook-test-components

The package is a collection of components for testing Storybook, implemented in React with TypeScript, and a custom workflow with GitHub Actions to publish the package on NPM and make a semantic release.

To use this package you need install it with:

```sh
yarn add ac-my-storybook-test-components
```
